# 线上考试系统

一个完整的线上考试系统，支持教师发布考试、管理学生、学生参加考试、自动批改和成绩查看。

## 快速启动

```bash
# 启动所有服务
docker-compose up --build -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down

# 重置数据库（清除所有数据重新初始化）
docker-compose down -v
docker-compose up --build -d
```

## 服务地址

| 服务 | 地址 | 说明 |
|------|------|------|
| 教师管理端 | http://localhost:8081 | 教师登录，管理考试、学生和查看成绩 |
| 学生考试端 | http://localhost:8082 | 学生登录，参加考试和查看成绩 |
| 后端API | http://localhost:8080 | Spring Boot 后端服务 |
| MySQL | localhost:3306 | 数据库服务 |

## 功能特性

### 教师端功能
- ✅ 考试管理：创建/编辑/删除/发布考试
- ✅ 题目管理：单选题、多选题、判断题
- ✅ 学生管理：添加/编辑/删除学生、重置密码
- ✅ 成绩查看：查看学生成绩、及格率、平均分统计

### 学生端功能
- ✅ 考试列表：查看可参加的考试
- ✅ 在线答题：支持倒计时、答题卡
- ✅ 考试规则：离开页面提示、自动提交
- ✅ 自动批改：提交后立即显示得分
- ✅ 成绩查看：查看历史成绩和答卷详情（含正确答案）

## 技术栈

- **后端**: Java 17 + Spring Boot 3 + MyBatis-Plus + MySQL 8.0
- **前端**: Vue 3 + Vite + Element Plus + Pinia + Axios
- **部署**: Docker + Docker Compose

## 项目结构

```
├── backend/                 # 后端服务
│   ├── src/main/java/com/exam/
│   │   ├── controller/     # 控制器
│   │   ├── service/        # 业务逻辑
│   │   ├── mapper/         # 数据访问
│   │   ├── entity/         # 实体类
│   │   ├── dto/            # 数据传输对象
│   │   └── config/         # 配置类
│   └── src/main/resources/
│       ├── schema.sql      # 数据库初始化脚本（含测试数据）
│       └── application.yml # 应用配置
├── frontend-admin/          # 教师管理端
├── frontend-user/           # 学生考试端
├── docker-compose.yml       # Docker编排
└── README.md
```

## 数据库初始化

数据库初始化脚本位于 `backend/src/main/resources/schema.sql`，包含：
- 数据库表结构（用户、考试、题目、选项、考试记录、答题记录）
- 测试用户数据（1个教师 + 3个学生）
- 示例考试数据（3套考试）
- 示例题目数据（25道题目及选项）

Docker 启动时会自动执行此脚本初始化数据库。
