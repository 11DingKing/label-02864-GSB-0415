## 详细流程说明

### 1. 出题阶段

1. **老师创建考试**：设置考试基本信息（标题、时长、及格分等）
2. **进入题目管理**：在题目管理页面点击"添加题目"
3. **选择题型**：在题型单选组中选择"简答题"（value=4）
4. **填写题目信息**：
   - 题目内容：输入简答题题干
   - 分值：设置该题分值
   - 参考答案：输入标准参考答案（供老师批改参考）
   - 关键词：输入评分关键词，多个关键词用英文逗号分隔（如：Java,Spring,SpringBoot）

### 2. 答题阶段

1. **学生进入考试**：选择考试并开始答题
2. **简答题展示**：简答题显示为多行文本输入框
3. **学生作答**：在文本框中输入完整答案
4. **提交试卷**：完成所有题目后提交

### 3. 自动评分阶段

系统提交后自动执行以下逻辑：

```java
private int calculateEssayScore(Question question, String userAnswer) {
    String[] keywords = question.getKeywords().split(",");
    int totalKeywords = keywords.length;
    int hitCount = 0;
    String lowerUserAnswer = userAnswer.toLowerCase();
    for (String keyword : keywords) {
        if (lowerUserAnswer.contains(keyword.trim().toLowerCase())) {
            hitCount++;
        }
    }
    return (int) Math.round(question.getScore() * ((double) hitCount / totalKeywords));
}
```

**算法说明**：
- 将学生答案转为小写（不区分大小写）
- 遍历所有关键词，统计命中数量
- 按命中比例计算得分：`得分 = 分值 × (命中数 / 总关键词数)`
- 四舍五入取整

### 4. 人工批改阶段

1. **老师查看记录**：在考试记录页面看到"待批改"状态的记录
2. **进入批改页面**：点击"批改"按钮进入评分界面
3. **批改信息展示**：
   - 题目内容和分值
   - 学生答案
   - 参考答案
   - 关键词命中情况（命中的关键词高亮显示）
   - 系统自动评分
4. **老师调整**：
   - 可修改最终得分（0到满分之间）
   - 可添加教师评语
5. **保存评分**：点击保存后，系统更新最终得分和考试状态

### 5. 数据结构说明

**Question 实体新增字段**：
- `referenceAnswer`：参考答案
- `keywords`：评分关键词

**AnswerRecord 实体新增字段**：
- `autoScore`：系统自动评分
- `finalScore`：老师最终评分
- `teacherComment`：教师评语

**ExamRecord 状态流转**：
- `0`：进行中
- `1`：已提交
- `2`：待人工批改（新增）
- `3`：已完成

## 核心特性

1. **自动评分**：基于关键词命中比例，减少老师批改工作量
2. **人工干预**：老师可随时调整分数，保证评分灵活性
3. **状态流转**：清晰的考试状态管理，便于追踪
4. **评语支持**：支持添加教师评语，反馈更详细
